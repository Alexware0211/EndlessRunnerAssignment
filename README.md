# Endless Runner Game

# Controls

These are the following controlls that you'll use

## Player 1 Controls
- **Space**: Jump
- **A Key**: Move left
- **D Key**: Move right

## Player 2 Controls
- **Up Arrow Key**: Jump
- **Left Arrow Key**: Move left
- **Right Arrow Key**: Move right
